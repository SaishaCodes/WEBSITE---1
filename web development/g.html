Sunny Bakery Challenge

Create a webpage for "Sunny Bakery" with the following elements:

  1. A heading with the bakery's name.
  2. A paragraph describing the bakery.
  3. An image of a loaf of bread.
  4. A list of available products.

Use the <h1> tag for the heading, <p> for paragraphs, and <img> for images. For lists, use <ul> and <li>.
